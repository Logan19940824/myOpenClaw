# Research & PPT Report Skill (美化版)

根据研究主题，联网搜索、分析总结，自动生成**专业美观**的 PPT 报告。

## ✨ 核心特点

- 🎨 **5 套专业主题** - 现代蓝、商务、科技、自然、渐变紫
- 📐 **渐变背景** - 封面和内容页都有精美的渐变效果
- ✨ **装饰元素** - 圆角、装饰条、页码等细节
- 🔍 **智能搜索** - 自动联网搜索相关信息
- 📊 **专业排版** - 清晰的标题、正文、要点层次
- 🌐 **多语言支持** - 中文、英文

## 📦 安装依赖

```bash
pip install python-pptx requests beautifulsoup4
```

## 🚀 使用方法

### 命令行使用

```bash
# 基本用法 (默认主题)
echo '{"topic": "人工智能发展趋势", "output": "ai-report.pptx"}' | python3 index.py

# 指定主题和页数
echo '{
  "topic": "新能源汽车市场分析",
  "output": "ev-report.pptx",
  "slides": 12,
  "theme": "business"
}' | python3 index.py
```

### 可用主题

| 主题 ID | 风格 | 配色 | 适合场景 |
|---------|------|------|----------|
| `modern-blue` | 现代蓝 | 蓝白渐变 | ✅ **默认**，通用场景 |
| `business` | 商务深蓝 | 深蓝+金色 | 正式商业报告 |
| `tech` | 科技风格 | 黑绿科技感 | 技术演示、产品发布 |
| `nature` | 自然绿 | 森林绿 | 环保、农业主题 |
| `gradient-purple` | 渐变紫 | 紫粉渐变 | 创意、艺术主题 |

## 📊 PPT 结构

每份 PPT 包含 **9 页** 专业内容：

| 页码 | 标题 | 内容 |
|------|------|------|
| 1 | 封面 | 主题 + 副标题 + 底部提示 |
| 2 | 目录 | 5 个主要章节 |
| 3 | 研究背景 | 目的、方法、数据来源 |
| 4 | 市场分析 | 规模、增长、参与者 |
| 5 | 发展趋势 | 技术、商业模式、政策 |
| 6 | 竞争格局 | 对手、份额、壁垒 |
| 7 | 挑战与机遇 | 痛点、风险、建议 |
| 8 | 未来展望 | 短期/中期/长期预测 |
| 9 | 总结 | 核心发现、决策建议 |

## 🎨 设计亮点

### 1. 渐变背景
- **封面**: 全屏渐变背景
- **内容页**: 顶部彩色装饰条 + 底部边框

### 2. 装饰元素
- 左侧蓝色竖条
- 右下角装饰圆圈
- 渐变半透明效果

### 3. 字体排版
- 大标题: 28pt，加粗
- 正文要点: 18pt
- 第一个要点: 20pt，加粗
- 页码: 12pt

### 4. 页码标注
- 每页右下角显示 "页码/总页数"

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| **python-pptx** | PPT 文件生成 |
| **MSO_SHAPE** | 形状绘制（矩形、椭圆） |
| **MSO_GRADIENT** | 渐变背景效果 |
| **requests** | HTTP 请求 |
| **BeautifulSoup** | HTML 解析 |

## 🐛 常见问题

### Q: 生成的 PPT 可以在 WPS 打开吗？
A: 可以，.pptx 是标准格式，WPS 和 Microsoft PowerPoint 都能打开。

### Q: 如何修改默认字体？
A: 在代码中修改 `THEMES` 字典的 `font_title` 和 `font_body` 值。

### Q: 支持自定义颜色吗？
A: 可以添加新的主题到 `THEMES` 字典中。

### Q: 可以导出 PDF 吗？
A: 当前版本生成 .pptx，可以在 PowerPoint 中另存为 PDF。
